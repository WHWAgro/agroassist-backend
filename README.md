### Instalaci√≥n

Instalar entorno conda:

```
conda env create -f environment.yml
```

y luego,

```
conda activate AgroAssistAPIRest
```

finalmente,

```
pip install -r requirements.txt
```

### Levantar servidor

Primero tienes que setear variables en el archivo env. 

luego

```
python app.py
```
o
```
flask run
```
